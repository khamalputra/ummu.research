🤔 Mengapa Macro F1?

Karena dataset tidak seimbang, Accuracy dapat menyesatkan. Macro F1 memberikan bobot yang sama pada setiap kelas tanpa memandang frekuensinya.

SLIDE 11

Uji Statistik McNemar

📋 Tujuan Pengujian

Menguji apakah perbedaan performa dua model bersifat signifikan secara statistik, bukan hanya kebetulan.

H₀: Tidak ada perbedaan signifikan antar model
H₁: Ada perbedaan signifikan (p < 0.05)
Exact Binomial Test
P = 2 × ∑i=0k C(n, i) ⋅ 0.5n

n = b + c  ·  k = min(b, c)

Dietterich (1998) Methodology
Matriks Kontingensi 2×2

Model B ✗
Model A ✓
